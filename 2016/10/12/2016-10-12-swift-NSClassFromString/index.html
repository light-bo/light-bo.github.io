NSClassFromString

从文档可以看出，这个 api 其实就是能够根据一个字符串获取该字符串对应的类的 class 对象，也就是说，这个 api 可以让我们通过提供一个类的类名（字符串），然后得到该字符串对应的类的实例。这在我们日常的开发中，在某些场景下，对我们简化代码是挺有用的一个东西。
